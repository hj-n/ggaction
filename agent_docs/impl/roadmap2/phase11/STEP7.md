# Roadmap 2 — Phase 11 Step 7: Gate B Public Regression Hierarchy

## 목표

Canonical unchanged user-facing regression chain이 approved primitive tree를 자동 생성하게 연결하고 full public
equivalence를 두 번째 Gate에서 검토한다.

## 진행 상태

- [x] Canonical public regression program with inferred attachment
- [x] Exact top-level and meaningful wrapped action hierarchy
- [x] Primitive/public semantic and resolved-scale equivalence
- [x] Primitive/public named tree and concrete property equivalence
- [x] Primitive/public Canvas-call and decoded-pixel equivalence
- [x] Resize, scale edit, component edit and highlight stability
- [x] `user-facing.png` and complete variant metadata
- [x] Gallery primitive/public pair and displayed call chain
- [x] Gate B code, image, tree and equivalence review package
- [ ] Explicit user approval before STEP8

## Gate B 결과

- Unchanged canonical public chain이 approved Gate A tree를 inferred attachment로 생성한다.
- `graphic-hierarchy/primitive.png`와 `user-facing.png`는 same-run decoded pixel hash가 같다.
- Canonical public/primitive program은 semantic state, resolved scales, named tree, concrete properties, Canvas calls와
  pixels가 모두 같다.
- Resize, x-scale edit, regression band/line edit와 point highlight 뒤에도 named draw order가 유지된다.
- Remaining chart-family primitive fixtures와 flat-order assertions는 의도적으로 STEP8 범위에 남겨 두었다. Gate B
  승인 전에는 이를 migration하지 않는다.

## Gate B

승인 전에는 hierarchy migration을 remaining chart families에 확장하거나 Phase closeout을 시작하지 않는다.

## 완료 조건

The unchanged public chain is visually approved and produces the same complete tree and pixels as the primitive oracle.
